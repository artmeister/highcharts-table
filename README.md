# Интерактивная таблица с графиками

## Описание

Интерактивная таблица данных с динамическим отображением графиков при выборе строки.

## Функциональность

1. **Таблица данных** - отображает показатели с текущими значениями, сравнением с вчерашним днем и этим днем недели
2. **Интерактивность** - при клике на строку таблицы отображается график с данными для выбранного показателя
3. **Графики** - реализованы с использованием библиотеки Highcharts

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- Highcharts

## Запуск

1. Откройте файл `index.html` в браузере
2. Или используйте локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (http-server)
   npx http-server
   ```
